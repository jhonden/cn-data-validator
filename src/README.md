# Application Source Code

This directory contains the main application files for the Network Package Validator.

## Files

- `validator_qt.py` - PyQt6 GUI main application (recommended)
- `validator_cli.py` - Command line interface
- `exceptions.py` - Custom exception classes

## Usage

```bash
# PyQt6 GUI (recommended)
python3 validator_qt.py

# CLI version
python3 validator_cli.py
```

**Note**: Development scripts have been moved to `scripts/` directory.
