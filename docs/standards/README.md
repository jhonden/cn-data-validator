### 开发过程

1. **开发前**：阅读相关约束和规范
2. **开发中**：遵循编码规范，使用中文注释
3. **开发后**：按照代码提交流程进行验证和提交

### 常见任务

- **新增功能**：参考项目约束中的功能范围
- **修复 Bug**：参考编码规范中的异常处理和日志规范
- **代码重构**：遵循编码规范，保持测试通过
- **更新文档**：更新对应规范文档或项目文档
