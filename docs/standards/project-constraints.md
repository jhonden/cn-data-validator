## 2. 功能范围约束

### ⚠️ 界面语言约束（必须严格遵守）

**约束等级**: ⭐⭐⭐⭐⭐ 一级约束（必须遵守）

**约束内容**：
- ❌ **禁止使用中文**：所有 GUI 界面文本必须使用英文
- ✅ **必须使用英文**：包括按钮、标签、提示、错误信息等
- ✅ **例外情况**：
  - 代码注释和文档可以使用中文（遵循开发规范）
  - 包类型识别结果可以用中文（如 "NIC包"）
  - 日志输出可以使用英文（便于问题排查）

**适用范围**：
- PyQt6 GUI (`validator_qt.py`)
- CustomTkinter GUI (`validator.py`)
- PySimpleGUI GUI (`validator_psg.py`)

**不适用范围**：
- 命令行界面 (`validator_cli.py`) - 可保持中文（便于国内用户）

**示例**：

✅ **正确做法**：
```python
# GUI 界面文本
self.select_btn = QPushButton("Select Directory")
self.validate_btn = QPushButton("Start Validation")
messageBox.warning("Warning", "Please select a data package directory first")
```

❌ **错误做法**：
```python
# GUI 界面文本
self.select_btn = QPushButton("选择目录")
self.validate_btn = QPushButton("开始校验")
messageBox.warning("警告", "请先选择数据包目录")
```

## 3. 性能约束

### 3.1 启动性能

- 程序启动时间不超过 3 秒
- GUI 界面加载时间不超过 1 秒

### 3.2 扫描性能

- 小型目录（<100 文件）：扫描时间 < 5 秒
- 中型目录（100-1000 文件）：扫描时间 < 30 秒
- 大型目录（>1000 文件）：扫描时间 < 2 分钟

### 3.3 内存使用

- 程序空闲时内存占用 < 100 MB
- 扫描大型目录时内存占用 < 500 MB

## 5. 安全约束

### 5.1 文件操作安全

- 解压文件前验证文件大小（避免解压炸弹）
- 限制解压深度和文件数量
- 使用临时目录，操作后自动清理

### 5.2 输入验证

- 所有用户输入（目录路径、文件路径）必须验证
- 验证路径存在性和有效性
- 验证文件格式支持的扩展名
